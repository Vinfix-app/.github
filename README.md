# VINFIX 🔍🚗

A real-time digital workspace and automated parts lookup system designed specifically for automotive mechanics and service stations (STO).

🚀 Overview
VINFIX moves beyond simple manual catalog searching. It provides technicians with a high-fidelity tool to scan vehicle codes, translate garage slang into verified OEM parts, and manage active client cards before they ever order a single component.

🛠 Tech Stack
- Framework: Next.js (React)
- Language: TypeScript
- Styling: Tailwind CSS
- Database/Backend: Supabase + Google AI Studio (Gemini API)

✨ Key Features
- Advanced Cinematic Scanner: Real-time mobile viewport with a frosted-glass mask optimized for 17-digit VIN capture.
- AI Slang Translation: Intelligent speech-to-text processing that converts informal garage jargon into engineering nomenclature.
- Seamless Garage CRM: Cloud-synchronized customer cards with live auto-save inputs for names and phone numbers.
- Supply Chain Matcher: Immediate downstream action link to cross-check part availability with active suppliers.
